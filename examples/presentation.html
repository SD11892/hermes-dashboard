HERMES

Bootstrap 4 Admin Dashboard

Overview

Huge Number of Components

The kit comes with components designed to look perfect together. All components fit perfectly with each other.

Multi-Purpose Sections

The sections will help you generate pages in no time. From team presentation to pricing options, you can easily customise and built your examples.

Example Pages

Speed up your development time or get inspired with the large number of example pages. You can jump start your development with our pre-built example pages.

Beautiful Notifications

Colourful notifications to animate your website

We have created multiple types of notifications. You can choose between different patterns (plain, with icon, with icon and close button) that comes in 4 different colours.

Awesome Tables

Awesome tables to get out of ordinary

We took into consideration multiple use cases and came up with some specific areas for you. If you need elements such as tables we've got you covered. They're beautiful and easy to use for the end user navigating your website.

Beautiful Gradients

Different colours on each page

Each page on the Hermes Dashboard has a different colour for the Page Header that fits perfect with the active colour on the sidebar.

Fully Responsive

Designed to be mobile first

HERMES Dashboard is built mobile-first and looks amazing on any device. Regardless of the screen size, the website content will naturally fit the given resolution.

Github Open Source

Contribute to it by making pull requests

HERMES Dashboard is an Open Source, so if you want to contribute to our project and make it better, your help is very welcome.

Custom Icons

Hermes Dashboard comes with 100 custom demo icons made by our friends from NucleoApp. The official package contains over 20.729 icons which are looking great in combination with Hermes Dashboard. Make sure you check all of them and use those that you like the most.
